Effects of Critic Scores on Video Game Sales

Fuel Economy of Electric Cars vs. Other Cars

This project looked at the difference in fuel economy and annual fuel costs of electric cars vs. other cars. There is a dashboard that reacts to user input and displays graphs dependent on the input compared to other metrics. There were also t-test run to determine if there is a statistical significant difference between the electric cars and other cars fuel economy and annual fuel costs.

This project was done using only Excel
